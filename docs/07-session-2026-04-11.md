### 1. 四个 Tab 的一键重置按钮

**涉及文件**：`useAppStore.ts`、`SourceManager.tsx`、`NodeManager.tsx`、`ProxyGroupEditor.tsx`、`RuleSetManager.tsx`

**交互设计**：两步确认，无弹框：
- 第一次点击 → 按钮变为**实心红色**，文案变为「确认重置？」
- 3 秒内第二次点击 → 执行重置
- 3 秒无操作 → 自动恢复原样

```ts
// 通用模式（各组件内）
const [confirmReset, setConfirmReset] = useState(false)
const confirmResetTimer = useRef<ReturnType<typeof setTimeout> | null>(null)

const handleResetClick = () => {
  if (confirmReset) {
    resetXxx()
    setConfirmReset(false)
    if (confirmResetTimer.current) clearTimeout(confirmResetTimer.current)
  } else {
    setConfirmReset(true)
    confirmResetTimer.current = setTimeout(() => setConfirmReset(false), 3000)
  }
}
```

```tsx
// 按钮 className 响应状态变化
className={`... transition-colors ${
  confirmReset
    ? 'bg-red-500 border-red-500 text-white hover:bg-red-600'
    : 'border-red-200 text-red-500 hover:bg-red-50'
}`}
```

**store 新增四个 reset action**：

| Action | 行为 |
|--------|------|
| `resetSources()` | `sources = []` |
| `resetManualProxies()` | `manualProxies = []` |
| `resetProxyGroups()` | 恢复为默认 PROXY + ♻️ 自动选择 两个分组 |
| `resetRules()` | 恢复为完整默认规则列表 |

### 3. 添加节点弹窗协议预填默认值

**涉及文件**：`NodeManager.tsx`

新增 `TYPE_DEFAULTS` 映射表，切换协议类型时自动填入端口等字段：

```ts
const TYPE_DEFAULTS: Partial<Record<ProxyType, Partial<FormState>>> = {
  socks5:    { port: '443',   tls: false, udp: true },
  http:      { port: '8080',  tls: false },
  ss:        { port: '8388',  cipher: 'aes-128-gcm' },
  vmess:     { port: '443',   alterId: '0', cipher: 'auto', network: 'tcp', tls: true },
  vless:     { port: '443',   network: 'tcp', tls: true },
  trojan:    { port: '443',   tls: true },
  hysteria2: { port: '443',   tls: true, up: '50', down: '200' },
  hysteria:  { port: '443',   up: '50', down: '200' },
  tuic:      { port: '443',   tls: true },
  anytls:    { port: '443',   tls: true },
  wireguard: { port: '51820', wgMtu: '1280', wgDns: '1.1.1.1' },
  ssh:       { port: '22' },
}
```

**关键点**：`EMPTY_FORM` 默认 `port: ''`，切换协议才触发 `TYPE_DEFAULTS`。新建弹窗初始时需单独应用默认值：

```ts
// 新建时（initial=null）合并初始类型的默认值
const [form, setForm] = useState<FormState>(
  initial ?? { ...EMPTY_FORM, ...(TYPE_DEFAULTS[EMPTY_FORM.type] ?? {}) }
)
```

类型切换时：
```tsx
onChange={(v) => {
  const t = v as ProxyType
  setForm((prev) => ({ ...prev, type: t, ...(TYPE_DEFAULTS[t] ?? {}) }))
}}
```

## 二、Bug 修复

### BUG-1：SourceManager 重置按钮 JSX 结构错误

**现象**：`tsc` 报 `Unexpected token`、`Expression expected`，错误出现在第 827+ 行（远离改动处）。

**根因**：增加重置按钮时，在 `<div className="flex items-center justify-between mb-3">` 内部增加了新的 `<div className="flex items-center gap-2">` 包裹层，但没有同时补上外层 `justify-between` div 的关闭标签 `</div>`。

**表现特征**：JSX 结构错误不在出错行本身报错，而是在"后续 TypeScript 函数里的 `=>`"（箭头函数）处报 `Unexpected token`，因为解析器以为仍在 JSX 上下文中，把 `>` 误读为 JSX 关闭标签。

**修复**：在 `hasRefreshableSources` 刷新按钮后补上 `</div>`。

```tsx
// 正确结构
<div className="flex items-center justify-between mb-3">
  <h2>...</h2>
  <div className="flex items-center gap-2">   {/* ← 内层 */}
    {重置按钮}
    {刷新按钮}
  </div>                                       {/* ← 关闭内层 */}
</div>                                         {/* ← 关闭外层（之前缺失） */}
```

**经验**：JSX 结构错误会导致 TS 报错位置与实际出错位置相差数百行，遇到 `Unexpected token '>'` 类错误，优先往上找未闭合的 JSX 标签。

### BUG-2：新建节点「端口不合法」报错

**现象**：打开「添加节点」弹窗后直接点击保存，报「端口不合法」。

**根因**：`TYPE_DEFAULTS` 只在**切换协议类型**时触发，但弹窗初始化时使用的是 `EMPTY_FORM`（`port: ''`），默认类型 socks5 的默认值未被应用。

**修复**：新建节点时合并初始类型的默认值（见功能 3）。

---

### BUG-3：Tooltip 被父容器 overflow-hidden 裁切

**现象**：鼠标悬停「⚠️ 不推荐」，tooltip 气泡上半部分被截断看不见。

**根因**：toolbar 所在父容器设置了 `overflow-hidden`（用于控制代码预览区的滚动边界）。`bottom-full` 定位的 tooltip 超出容器顶边会被裁切。

**修复**：改为 `top-full` 向下弹，箭头用 `border-b-*` 朝上（见功能 5）。

**通用结论**：绝对定位的 tooltip 弹出方向必须考虑最近的 `overflow-hidden` 祖先，朝向有内容空间的方向弹。

### BUG-4：dialer-proxy 示例配置方向错误

**现象**：链式代理 YAML 示例中，`dialer-proxy` 加在了「香港节点」上，实际出口 IP 是香港而非美国。

**根因**：`dialer-proxy` 的语义是「**连接到本节点服务器时**，先经过指定代理建立底层连接」。加在香港节点上 = 连接香港服务器时走美国SOCKS5，出口在香港；加在美国SOCKS5上 = 连接美国服务器时走香港节点，出口在美国。

**修复**：

```yaml
# 错误（出口 = 香港）
- name: "🇭🇰 香港节点"
  type: vmess
  dialer-proxy: "🇺🇸 美国住宅SOCKS5"   # ← 错

# 正确（出口 = 美国住宅）
- name: "🇺🇸 美国住宅SOCKS5"
  type: socks5
  dialer-proxy: "🇭🇰 香港节点"          # ← 正确

proxy-groups:
  - name: "🇺🇸 美国原生出口"
    proxies:
      - "🇺🇸 美国住宅SOCKS5"            # ← 代理组选 SOCKS5，不是香港节点
```

**链路对比**：

| 写法 | 链路 | 出口 IP |
|------|------|---------|
| dialer-proxy 在香港节点 | 本地 → SOCKS5 → 香港 → 目标 | 香港 ✗ |
| dialer-proxy 在SOCKS5 | 本地 → 香港 → SOCKS5 → 目标 | 美国住宅 ✓ |
